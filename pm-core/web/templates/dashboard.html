About “Age”
Age = elapsed time since the last received report for that host (from processed_utc).
- Shorter age = fresher data.
- Large age may indicate Offline condition.
Legend / เกณฑ์สถานะ (ตัวอย่างไอคอนจริงในระบบ)
  • 🟢 Up — เครื่องออนไลน์ ทำงานปกติ (เช่น CPU/Mem/Disk ต่ำกว่า ~75%)
  • 🟡 Warning — มีสัญญาณเสี่ยง (ค่าสูงสุดของ CPU/Mem/Disk ~75–89%)
  • 🔴 Degraded — ภาวะวิกฤต/เสื่อม (ค่าสูงสุด ≥ ~90%)
  • ⚪ Offline — ไม่ได้รับรายงานเกินระยะเวลาที่กำหนด (Age ยาวผิดปกติ)
